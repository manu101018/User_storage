List of REST API:
1. http://localhost:{port}/auth/signup                       - To signup new user                       (method:POST)
2. http://localhost:{port}/auth/login                        - To login                                 (method:POST)
3. http://localhost:{port}/user-list                         - To get the list of user                  (method:GET)
4. http://localhost:{port}/details?userId={userId}           - To get the details of user               (method:GET)
5. http://localhost:{port}/upload?userId={userId}            - To upload the .csv file of product       (method:POST)
6. http://localhost:{port}/product-list?userId={userId}      - To get the list of products              (method:GET)
